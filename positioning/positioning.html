Positioning

This is a css style property used to define where the item lies and modify its position.

It has 5 values:

  • static
  • relative
  • fixed
  • sticky
  • absolute

STATIC

This is the default property and is applied automatically to all elements when position isn't mentioned.

RELATIVE

When an element is set to have relative position one can vary the location of the element relative to the position it would have had by default.

FIXED

This let's us fix the position of the element somewhere on the screen and it will stay there even when you scroll. This element will be rendered in a different layer and thus overlaps the other elements. You can manage the ovelaps using the z-index property.

STICKY

This is a secret lovechild of static and fixed, because it acts like a static element at first but the once you scroll down it and it hits the position you set using(top, left, right and/or bottom) it will become like fixed and stick to your screen and persist with scroll at the specified position. Scroll back to it's original position and it turns back to it's static personality.

ABSOLUTE

This is similar to relative but instead of offsetting the element based on it normal position this will position the element based of it's nearest ancestor element that has a defined position. If none of it's parent elements have a position property then it will be offset with respect to the body element.
